## MVP 범위

### CRUD 4종 — 화면에서 모두 동작

모든 생성·조회·수정·삭제는 별도 페이지 이동 없이 **화면 안에서 직접** 수행한다.

| 동작 | 설명 |
|------|------|
| **추가** | 태스크 생성 폼 (인라인 또는 모달) |
| **목록** | 전체 태스크 목록, 상태별 필터 |
| **수정** | 제목·설명·상태·담당자·마감시각 인라인 편집 |
| **삭제** | 확인 다이얼로그 후 즉시 반영 |

### 상태 분류

```
todo  →  in_progress  →  review  →  done
```

### 마감시각 (due_at)

- 날짜만이 아닌 **시간까지** 지정·표시한다.
- 형식 예시: `2026-05-12 18:00`
- DB 저장: ISO 8601 UTC (`2026-05-12T09:00:00.000Z`)
- UI 표시: 로컬 타임존 기준 `YYYY-MM-DD HH:mm`
- 마감 1시간 이내: 주황색 강조 / 마감 초과: 빨간색 강조

### 라이트 / 다크 테마 토글

- 헤더에 토글 버튼 노출
- 선택값은 `localStorage`에 저장 → 새로고침 후에도 유지
- 시스템 설정 감지(`prefers-color-scheme`)를 초기값으로 사용
- SSR 시 깜빡임(flash) 방지: 인라인 스크립트로 HTML 클래스 선적용

### 모바일 반응형

- 최소 지원 너비: **360px**
- 360px에서 레이아웃 깨짐 없음이 성공 기준
- 터치 타겟 최소 44×44px (WCAG 2.5.5)
- 모바일에서 인라인 편집 대신 바텀시트(bottom sheet) 사용 가능

## UI 톤 & 스타일

> **macOS 스타일** — 군더더기 없고 부드럽고 정돈된 느낌

| 요소 | 규칙 |
|------|------|
| **모서리** | `border-radius: 12px` 기본, 소형 요소 `8px` |
| **그림자** | 부드러운 다중 레이어 그림자 (`box-shadow: 0 2px 8px rgba(0,0,0,0.08), 0 8px 24px rgba(0,0,0,0.06)`) |
| **카드** | 반투명 배경 (`backdrop-filter: blur(12px)`, `background: rgba(255,255,255,0.72)`) |
| **폰트** | 시스템 폰트 스택 (`-apple-system, BlinkMacSystemFont, 'Segoe UI', sans-serif`) |
| **색상** | 라이트: 흰색 기반 / 다크: `#1c1c1e` 기반 (iOS 다크모드 참고) |
| **간격** | 4px 그리드 기반 (8, 12, 16, 24, 32px) |
| **애니메이션** | `transition: 150ms ease` 기본, 모달 등장 `200ms ease-out` |
| **아이콘** | lucide-react (선형, 2px stroke) |

## 확장 로드맵 (MVP 이후)

> MVP 완료 후 우선순위에 따라 단계적으로 추가한다. 현재는 설계에 영향을 주는 수준만 고려한다.

| 단계 | 기능 |
|------|------|
| 확장 1 | JWT 로그인 — 이메일/비밀번호 인증, 역할 기반 접근 제어 |
| 확장 2 | 팀 관리 — 멤버 초대, 프로젝트별 멤버 배정 |
| 확장 3 | Kanban 보드 — 드래그 앤 드롭 상태 변경 |
| 확장 4 | 채팅 — 태스크 내 댓글 스레드 |
| 확장 5 | CI/CD — GitHub Actions 자동 배포 파이프라인 |

| WebSocket (실시간) | **MVP는 폴링으로 구현**, 확장 단계에서 SSE 또는 WebSocket 전환 |
